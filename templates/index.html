🔥 Algerian Forest Fires Dataset

📊 Dataset Overview

The dataset includes 244 instances collected from two Algerian regions:

  • Bejaia (northeast) and Sidi Bel-abbes (northwest)
  • Each region contains 122 instances (total: 244)
  • Time period: June to September 2012
  • Includes 11 attributes + 1 target output (Fire Weather Index - FWI)
  • Classes: 138 fire and 106 no-fire cases

📌 Attribute Details

  1. Date (DD/MM/YYYY) — June to September 2012
  2. Temperature (°C): 22 to 42
  3. Relative Humidity (%): 21 to 90
  4. Wind Speed (km/h): 6 to 29
  5. Rain (mm): 0 to 16.8
  6. FFMC index: 28.6 to 92.5
  7. DMC index: 1.1 to 65.9
  8. DC index: 7 to 220.4
  9. ISI index: 0 to 18.5
  10. BUI index: 1.1 to 68
  11. FWI index: 0 to 31.1
  12. Classes: 0 = Not Fire, 1 = Fire
